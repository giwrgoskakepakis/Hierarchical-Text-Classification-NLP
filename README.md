# Hierarchical-Text-Classification-NLP

Classification is one the the fundamental utilizations of machine learning models such as neural networks. In this project we are called to create a neural network
model to perform hierarchical classification of 2 levels for texts, and specifically news documents. The dataset we use (’news-classification.csv’) consists of 10917
text samples, which are classified in 17 main classes (’category level 1’) and 109 total sub-classes (’category level 2’). This project was developed within the
framework of the course of ’Neuro-Fuzzy Computing’ (ECE-447) for University of Thessaly in February/March 2024.
